# 变更：dApp 运维入口与体验完善（参考 Nouns）

## 为什么
MVP 已可用，但缺少运维/可观测入口与交易追踪体验。参考 Nouns dApp 的“清晰状态 + 可追踪 + 管理入口”风格，补齐关键 UX 与运维操作，提升可用性与可维护性。

## 变更内容
- dApp 增加交易哈希、合约地址的区块浏览器链接展示
- Admin 页面新增暂停/恢复（Pausable）入口
- 增加流程状态时间线或步骤提示（投票/交付/质疑/结算）
- 增加全局可观测信息：链 ID、RPC 状态、当前区块时间
- 文档补充运维与用户快速指引（延续 Nouns 风格）

## 影响范围
- 影响规格：dapp、docs
- 影响代码：dapp/app/**、dapp/lib/**、docs/*
