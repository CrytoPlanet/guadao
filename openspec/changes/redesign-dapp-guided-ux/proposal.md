# 变更：dApp 傻瓜式引导与界面重设计

## 为什么
当前前端信息密度高、流程分散，对新手不友好且视觉体验一般。需要更直观的引导、清晰分区与美观的视觉系统，降低操作门槛并提升信任感。

## 变更内容
- 统一视觉语言与版式，提高整体美观度与一致性
- 以任务流为核心：将复杂操作拆解为步骤式/引导式
- 对关键术语提供简短解释或提示
- 关键操作尽量自动填充与兜底校验，减少手动输入
- 将管理/高级功能收纳至“高级模式”，默认隐藏

## 影响范围
- 影响规格：dapp
- 影响代码：dapp/app/**、dapp/app/components/**、dapp/app/globals.css、dapp/lib/i18n.js
